# Positivus Landing Page

Адаптивный лендинг, созданный с использованием **HTML**, **SCSS** и **JavaScript**. Проект демонстрирует современную структуру верстки, семантическую разметку и организацию стилей с препроцессором SCSS.

## О проекте

Этот репозиторий содержит код лендинга. Основная цель проекта — отработка навыков верстки по методологии (судя по структуре файлов) и создания адаптивных интерфейсов. В проекте реализованы:

*   Семантическая верстка HTML5.
*   Стилизация с использованием препроцессора SCSS, разделение на компоненты (стили для header, footer и других секций).
*   Адаптивность для различных устройств.
*   Базовая интерактивность на JavaScript.

## Технологии

*   **HTML5**: для структуры и контента.
*   **SCSS (Sass)**: для написания модульных и поддерживаемых стилей (скомпилируется в обычный CSS).
*   **JavaScript**: для реализации интерактивных элементов.

Скачайте архив с проектом или клонируйте репозиторий:
```bash
git clone https://github.com/VaganovaIP/positivus-landing-scss.git

Откройте папку с проектом:
```bash

Запустите сайт — просто дважды кликните по файлу index.html или откройте его через браузер.
